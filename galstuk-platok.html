Шейные аксессуары

Контекст использования трёх главных мужских аксессуаров.

Шейный аксессуар — последняя деталь, которая закрывает образ. Три варианта: галстук, платок-pochette (нагрудный платок), бабочка. У каждой свой контекст.

Галстук — самый универсальный, для деловых встреч, переговоров, формальных мероприятий. Однотонный шёлк-grenadine — рабочая лошадка, не выйдет из моды. Платок-pochette — для неформального дня, добавляет цвет в нагрудный карман, не должен «матчить» галстук цветом.

Бабочка (bow tie) — для смокинга и редких очень формальных событий. Не для офиса, не для свадьбы (если только не black-tie). Подробнее про контекст ношения — Style Notes — гид по стилю.
